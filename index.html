Introducción a JavaScript

¿Qué es JavaScript?

JavaScript es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico.

Se utiliza principalmente del lado del cliente, implementado como parte de un navegador web permitiendo mejoras en la interfaz de usuario y páginas web dinámicas, aunque existe una forma de JavaScript del lado del servidor.

JavaScript se diseñó con una sintaxis similar a C, aunque adopta nombres y convenciones del lenguaje de programación Java. Sin embargo, Java y JavaScript tienen semánticas y propósitos diferentes.

Lenguaje Interpretado

JavaScript es un lenguaje interpretado, lo que significa que no necesita compilación. Los navegadores web interpretan el código directamente, facilitando el desarrollo y la depuración.

Orientado a Objetos

JavaScript utiliza un modelo de objetos basado en prototipos, en lugar del modelo basado en clases más común. Los objetos pueden heredar propiedades directamente de otros objetos.

Historia de JavaScript

JavaScript fue creado por Brendan Eich en 1995 mientras trabajaba en Netscape Communications Corporation. Inicialmente se llamó Mocha, luego LiveScript, y finalmente JavaScript.

Fue influenciado por muchos lenguajes como Self, Scheme, Java y C. A pesar del nombre, JavaScript no está relacionado con Java, aunque su sintaxis es similar.

En 1997, JavaScript fue estandarizado por ECMA International como ECMAScript, lo que permitió su evolución controlada y su adopción generalizada.

Elementos de JavaScript

Variables

Las variables se utilizan para almacenar valores de datos. En JavaScript puedes usar var, let y const para declarar variables.

let nombre = "Juan";
const edad = 30;

Tipos de Datos

JavaScript tiene tipos de datos primitivos y objetos. Los primitivos incluyen string, number, boolean, null, undefined y symbol.

let texto = "Hola";
let numero = 42;
let booleano = true;

Operadores

JavaScript incluye operadores aritméticos, de asignación, comparación, lógicos, bit a bit, de cadena y ternarios.

let suma = 5 + 3;
let mayor = 10 > 5;
let resultado = (edad >= 18) ? "Adulto" : "Menor";

Estructuras de Control

Permiten controlar el flujo de ejecución del código. Incluyen if/else, switch, for, while, do-while y try/catch.

if (condicion) {
  // código
} else {
  // otro código
}

Funciones

Las funciones son bloques de código diseñados para realizar una tarea particular. Pueden recibir parámetros y devolver valores.

function sumar(a, b) {
  return a + b;
}

Objetos

Los objetos son colecciones de propiedades, donde cada propiedad es una asociación entre un nombre (clave) y un valor.

let persona = {
  nombre: "Aren",
  edad: 28
};
